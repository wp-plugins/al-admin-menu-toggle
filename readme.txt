== Description ==

This Plugin is used for changing WordPress Default Hover Menu to Toggle Menu.

Easy to Use. Just Install and Activate. 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `admin-menu-toggle` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

If Multi-site:

1.	Follow these steps to install this plugin. 
Login -> Dashboard -> Network Admin -> Plugins -> Add Plugin -> Upload.

2.	Upload the zipped folder (admin-toggle-menu.zip) using the default file uploads. Then click on Network Activate to Activate in all the sub-sites.

3.	To specifically install in specific blog, go to the specific site dashboard-> Installed Plugins -> find the Plugin “AL | Admin Menu Toggle” and click on Activate.
